# Pandas Cheat Sheet for Data Cleaning

## 1. Loading Data
```python
import pandas as pd

# Load CSV file
df = pd.read_csv('filename.csv')

# Display first 5 rows
print(df.head())

# Display info about columns and data types
print(df.info())
```

## 2. Handling Missing Values
```python
# Check for missing values
print(df.isnull().sum())

# Drop rows with missing values
df_dropped = df.dropna()

# Fill missing values with a specific value (e.g., 0)
df_filled = df.fillna(0)

# Fill missing values with the mean of the column
df['column_name'] = df['column_name'].fillna(df['column_name'].mean())
```

## 3. Removing Duplicates
```python
# Check for duplicates
print(df.duplicated().sum())

# Remove duplicates
df = df.drop_duplicates()
```

## 4. Data Type Conversion
```python
# Convert column to numeric (errors='coerce' turns non-numeric values into NaN)
df['column_name'] = pd.to_numeric(df['column_name'], errors='coerce')

# Convert column to integer
df['column_name'] = df['column_name'].astype(int)
```

## 5. Filtering and Cleaning
```python
# Filter rows based on a condition
df_filtered = df[df['score'] > 0]

# Replace specific values
df['grade'] = df['grade'].replace('missing', pd.NA)
```

## 6. Saving Data
```python
# Save to a new CSV file
df.to_csv('cleaned_data.csv', index=False)
```
